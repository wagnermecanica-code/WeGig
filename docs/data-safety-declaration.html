Declaração de Segurança de Dados

O WeGig está comprometido em proteger a privacidade e segurança dos dados de nossos usuários. Esta declaração detalha nossas práticas de coleta, uso e proteção de dados pessoais, conforme exigido pelo Google Play Console.

Dados Coletados

Dados Fornecidos pelo Usuário

  • Email: Para autenticação e comunicações
  • Nome do perfil: Para identificação na plataforma
  • Informações musicais: Instrumentos, gêneros, nível de habilidade
  • Localização: Cidade e coordenadas geográficas
  • Fotos do perfil: Imagem opcional do usuário
  • Conteúdo de posts: Texto e imagens compartilhadas

Dados Coletados Automaticamente

  • Dados de uso: Interações no app, tempo de uso
  • Dados técnicos: Versão do app, tipo de dispositivo
  • Dados de localização aproximada: Para funcionalidades de busca

Finalidade do Uso dos Dados

  • Autenticação: Verificar identidade dos usuários
  • Funcionalidades principais: Conectar músicos e bandas
  • Busca geolocalizada: Encontrar profissionais próximos
  • Notificações: Alertar sobre oportunidades relevantes
  • Comunicação: Mensagens entre usuários
  • Segurança: Prevenir fraudes e abusos
  • Melhorias: Otimizar a experiência do usuário

Compartilhamento de Dados

Não Compartilhamos com Terceiros

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros para fins comerciais.

Compartilhamento Necessário

  • Serviços Google: Firebase para armazenamento e autenticação
  • Google Maps: Para funcionalidades de geolocalização
  • Autoridades legais: Apenas quando exigido por lei

Segurança dos Dados

  • Criptografia: Todos os dados em trânsito e armazenados são criptografados
  • Controles de acesso: Apenas pessoal autorizado acessa dados
  • Auditorias regulares: Revisões de segurança frequentes
  • Backups seguros: Dados protegidos contra perda

Direitos dos Usuários

Os usuários têm direito a:

  • Acesso: Ver todos os dados que temos sobre eles
  • Correção: Atualizar informações incorretas
  • Exclusão: Remover conta e dados associados
  • Portabilidade: Exportar dados em formato legível
  • Revogação: Cancelar consentimento para certas coletas

Contato sobre Privacidade

Para exercer seus direitos ou tirar dúvidas sobre privacidade:

  • Email: privacidade@wegig.com.br
  • Política completa: Política de Privacidade

Conformidade Legal

Cumprimos com as leis de proteção de dados aplicáveis:

  • Brasil: Lei Geral de Proteção de Dados (LGPD)
  • União Europeia: Regulamento Geral de Proteção de Dados (GDPR)
  • Estados Unidos: Lei de Privacidade da Califórnia (CCPA)

Transparência e Atualização

Mantemos esta declaração atualizada e transparente sobre nossas práticas. Qualquer mudança significativa será comunicada aos usuários e atualizada nesta página.
